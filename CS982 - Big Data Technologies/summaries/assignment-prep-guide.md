## üìã Assignment Overview

### Part I: Create (70% of grade)
**Deliverables:**
- Report: 2500 words (+/- 10%) as PDF
- Code: All analysis code submitted

**Task:** Analyze an open dataset and write a data science report

**Key Sections:**
1. Problem identification (10%)
2. Dataset introduction (10%)
3. Summary statistics with figures (20%)
4. ONE unsupervised learning method (20%)
5. ONE supervised learning method (20%)
6. Reflection on methods (10%)
7. Structure & references (10%)

### Part II: Evaluate (30% of grade)
**Task:** Self-assess Part I and justify your mark
- Give Part I a mark (0-100)
- Write 1000-word justification
- Your mark depends on how close you are to marker's grade

## üö® Common Mistakes to Avoid

### Dataset Selection
‚ùå Using banned datasets (Iris, Titanic, Wine, etc.)
‚ùå Using datasets from course labs/lectures
‚úÖ Find unique dataset on Kaggle or UCI ML Repository

### Analysis Depth
‚ùå Just showing code output without explanation
‚ùå Tables/figures in appendix instead of main text
‚ùå No rationale for method chosen
‚úÖ Clear explanation of WHY you chose each method
‚úÖ Interpret results in context of your question
‚úÖ Justify metrics used

### Writing Style
‚ùå Too technical with no plain English explanation
‚ùå Too casual/informal
‚ùå No references
‚úÖ Balance technical accuracy with clarity
‚úÖ Academic tone but readable
‚úÖ Cite data sources and methods

### Reflection Section
‚ùå "Everything worked great, no issues"
‚ùå Too brief (2-3 sentences)
‚úÖ Honest evaluation of limitations
‚úÖ Specific suggestions for improvements
‚úÖ Show critical thinking
